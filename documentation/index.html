About BioActive

The goal of the BioActive project is to develop an open, community standard for running Automated Science campaigns using artificial intelligence and laboratory automation. BioActive provides a structure for defining an experimental space specification (ESS) consisting of specified independent, dependent and associated variables, a predictive model construction function, an active learning function, and an objective function tethered to a termination criteria. It facilitates active learning experimentation, delivering a model for the aforementioned space. This framework is comprised of minimally constraining components and is capable of producing reliable models while reducing unnecessary experimentation.
