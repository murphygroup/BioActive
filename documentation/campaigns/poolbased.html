BioActive Pool-Based Classification Campaigns

This type of campaign defines an Experimental Space consisting of a single independent variable and a single dependent variable. The independent variable is an index into a pool of samples, each of which is described by "associated variables" (features). The dependent variable is a class label. The predictive modeling trains a classifier to predict the class label for each value of the independent variable from its associated variables. The active learner chooses samples to label using a "Query By Committee" approach in which available labeled data are fit using different classification methods and the samples showing the most disagreement among the classifiers are prioritized for labeling.
