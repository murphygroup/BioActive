About BioActive Discrete Variable Campaigns

This type of campaign defines an Experimental Space of a specified number of independent variables and sets of discrete values that each independent variable can take on, and a single dependent variable. Its predictive model is a multivariate linear regression module, and it uses uncertainty sampling to choose combinations of the independent variables to sample.
