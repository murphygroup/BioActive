About createCampaign's Structure and Design

The createCampaign file exists for all campaigns and provides the definitions for the following functions:

Initialization

def initAvars(self)

This function initializes associated variables for the pool-based campaign. Imports from the "predictive_modelers" folder. This function is called once at the beginning of runCampaign2.py. For other campaigns that do not require this, simply return.

Goal Tether

def goalTether(self, arg)

This function is used to determine whether the goal has been reached or not (i.e. used to decide whether to continue experimenting for the current iteration or not). Imports from the "goal_tether_functions" folder. Further information can be found in the "Goal Tether" page.

Data Modeler

def modelData(self, *arg)

This function is used to model the experimental space, store necessary information such as accuracy, confidence and predictive modeling function. Imports from the "predictive_modelers" folder. Further information can be found in the "Modeler" page.

Active Learner

def activeLearner(self, arg)

This function selects predictions with the lowest confidence/highest uncertainty to get data for in the next iteration. Imports from the "active_learners" folder. Further infromation can be found in the "ActiveLearner" page.

Fetch Data

def fetchData(self, arg1, arg2)

This function simulates grabbing data from a scientific instrument. Imports from the "data_acquisition" folder. Further information can be found in the "Data Acquisition" page.

ESS Class

The ESS class defines the experimental space by specifying what the independent and dependent variables are.

Provide Names

def provideNames(self)

This function simply prints the names of each independent variable as well as the names of the categories if the independent variables are categorical.
